# BOJ 13549 숨바꼭질 3
### [문제 바로가기](https://www.acmicpc.net/problem/13549)
| 난이도 | 정답률 | 분류 |
| ------ | ------ | ------ |
|  Gold V | 24.720% | 0-1 BFS , Dijkstra |


### 구현
---
수빈이가 동생을 찾는 가장 빠른 시간을 구하는 문제로, 이동 방식에 따라 가중치(소요 시간)가 다르다는 점이 핵심이다.

* **순간이동 ( 2 * X ):** 0초 소요
* **걷기 ( X - 1 , X = 1):** 1초 소요

가중치가 0과 1로 구성되어 있으므로, 일반적인 BFS 대신 **0-1 BFS**를 구현하기 위해 `Deque`를 활용했다.


### 결과
---
16,020KB	/ 88ms


### 마치며
---
가중치가 다른 그래프 탐색에서 최단 거리를 구할 때, `PriorityQueue`를 사용하는 Dijkstra 알고리즘도 가능하지만, 가중치가 0과 1뿐이라면 `Deque`를 이용한 **0-1 BFS**가 $O(V+E)$의 시간 복잡도로 더 효율적일 것 같아 0-1 BFS 알고리즘을 사용했다.
