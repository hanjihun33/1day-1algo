# BOJ 1967 트리의 지름
### [문제 바로가기](https://www.acmicpc.net/problem/1967)
| 난이도 | 정답률 | 분류 |
| ------ | ------ | ------ |
|  Gold IV | 41.358% | dfs |


### 구현
---
루트가 1번인 가중치 트리가 주어진다.
트리의 지름을 구하는 문제이다.

### 결과
---
20,264KB	/ 140ms


### 마치며
---
트리의 지름은 시작점 기준 최장거리가 아니라,
트리 전체에서 가장 먼 두 점 사이의 거리이기 때문에
DFS를 두 번 수행해야 한다는 점이 핵심이었다.
