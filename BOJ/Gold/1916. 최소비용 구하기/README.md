# BOJ 1916 최소비용 구하기
### [문제 바로가기](https://www.acmicpc.net/problem/1916)
| 난이도 | 정답률 | 분류 |
| ------ | ------ | ------ |
|  Gold V | 34.000% | dijkstra |

### 구현
---
N개의 도시(정점)와 M개의 버스(간선) 정보가 주어질 때, 출발 도시에서 도착 도시까지 가는 **최소 비용(최단 경로)**을 구하는 문제입니다. 간선의 가중치가 0 이상이므로 다익스트라(Dijkstra) 알고리즘을 사용했습니다.

### 결과
---
52,080KB	/ 428ms

### 마치며
---
처음엔 단순히 큐를 사용하는 BFS로 접근할 수도 있지만, 간선마다 가중치(버스 비용)가 다르기 때문에 최단 거리를 보장하려면 다익스트라 알고리즘이 필수적이라는 것을 확인할 수 있었습니다.
