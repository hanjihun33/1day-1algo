# BOJ 9251 LCS
### [문제 바로가기](https://www.acmicpc.net/problem/9251)
| 난이도 | 정답률 | 분류 |
| ------ | ------ | ------ |
|  Gold IV | 42.591% | dp |


### 구현
---
두 수열이 주어졌을 때, 모두의 부분 수열이 되는 수열 중 갖아 긴 것을 찾는 문제다.

2차원 DP 테이블을 활용해 문제를 해결했다.

문자가 같을 때 ($a[i] == b[j]$):이전 공통 부분 수열에 현재 문자를 추가한다.$$dp[i][j] = dp[i-1][j-1] + 1$$문자가 다를 때 ($a[i] \neq b[j]$):기존에 찾은 최대 길이를 유지한다. (위쪽 혹은 왼쪽 값 중 큰 값 선택)$$dp[i][j] = \max(dp[i-1][j], dp[i][j-1])$$

### 결과
---
16,044KB	/ 80ms


### 마치며
---
처음에는 한 글자씩 비교하는 브루트포스를 생각했으나,이 문제에서는 시간 초과가 불가피하다는 것을 알게 되었다.

이를 해결하기 위해 DP를 도입하여 최적화할 수 있었다. 현재는 2차원 배열을 사용해 직관적으로 풀이했지만, 이전 행의 정보만 참조한다는 점을 이용해 1차원 배열로 개선한다면 메모리 효율을 더욱 높일 수 있을 것 같다.
