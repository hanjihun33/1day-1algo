# BOJ 1541 잃어버린 괄호
### [문제 바로가기](https://www.acmicpc.net/problem/1541)
| 난이도 | 정답률 | 분류 |
| ------ | ------ | ------ |
|  Silver II | 55.810% | greedy |


### 구현
---
이 문제의 핵심은 **"가장 큰 수를 빼서 전체 결과값을 최소로 만드는 것"**이다. 뺄셈(-) 부호를 기준으로 괄호를 적절히 치면, 그 뒤에 나오는 모든 더하기(+) 연산을 먼저 수행하여 뺄셈의 크기를 최대화할 수 있다.

### 결과
---
11,528KB	/ 60ms


### 마치며
---
'-' 를 기준으로 식을 나누고, 나머지 식들을 뺀다.

자바의 split 메서드를 사용할 때 + 기호는 정규표현식 예약어이므로 \\+로 이스케이프 처리를 해주는 것을 배웠다.
